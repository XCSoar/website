XCSoar is a tactical glide computer that provides glider pilots with comprehensive flight planning, navigation, and analysis tools.

  • Moving map with terrain, topography, and airspace visualization
  • Real-time flight calculations and navigation assistance
  • Thermal assistance for finding and centering thermals
  • FLARM traffic integration for collision avoidance
  • Task planning with support for AAT, racing, and cross-country tasks
  • Flight analysis with barograms, climb history, and performance metrics
  • Multi-platform - runs on Android, Windows, and Unix systems

Whether you're flying cross-country, competing in competitions, or training, XCSoar offers the tools you need to make informed decisions in the air.
