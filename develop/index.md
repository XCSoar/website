## Contributing

XCSoar is a free software project run by volunteers. Here's how you can help:

- **Core development**: C++ developers can [get the source code](https://xcsoar.readthedocs.io/en/latest/), make changes, and submit patches. [Contact developers](/contact/) or [pick an issue](https://github.com/XCSoar/XCSoar/issues) to work on.
- **Testing**: Test [released versions](https://xcsoar.org/download/) or [testing versions](https://download.xcsoar.org/testing/) (**ground only** - unstable) and [report bugs](/develop/new_ticket.html).
- **Translation**: Help improve translations in [30+ languages](https://hosted.weblate.org/projects/xcsoar).
- **Documentation**: Edit the manual (LaTeX) or improve the [website](https://github.com/XCSoar/website/issues).
- **Data management**: Help with [map generation](https://github.com/XCSoar/xcsoar-mapgen-data/issues), [data repository](https://github.com/XCSoar/xcsoar-data-repository/issues), or [data content](https://github.com/XCSoar/xcsoar-data-content/issues).
